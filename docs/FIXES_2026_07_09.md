# Critical Fixes Summary - 2026-07-09

## 🚨 Five Critical Issues Fixed

### P0: Annotation Parser Missing Most Seizures
**Impact**: Severe data corruption - many seizure files would be entirely mislabeled as normal

**Fixed**: `src/utils/annotation_parser.py`
- Now supports both "Seizure Start Time" and "Seizure N Start Time" formats
- Tested on chb06_01: correctly finds 3 seizures (was returning [] before)

### P1: No Batch Processing Support
**Impact**: Cannot process full dataset for Step 6

**Fixed**: `scripts/01_create_windows.py`
- Added argparse with `--all-patients`, `--all-files`, `--patient`, `--file`
- Automatically finds files with seizures
- Usage: `python scripts/01_create_windows.py --all-patients`

### P1: Missing Critical Metadata
**Impact**: Cannot do patient-level split, cannot track augmented windows, cannot compute seizure-level metrics

**Fixed**: `src/preprocessing/windowing.py`
- Now saves complete metadata including:
  - `patient_id` (for train/test split)
  - `window_metadata` (per-window timing, augmentation info)
  - `seizure_intervals` (for seizure-level evaluation)
  - `random_seed` (for reproducibility)

### P2: FZ-CZ Fallback Not Working
**Impact**: Data loading fails when FZ-CZ unavailable

**Fixed**: `src/preprocessing/windowing.py`
- Now actually uses the alternative channel found
- Reloads data with adjusted target channels

### P2: Non-reproducible Augmentation
**Impact**: Cannot reproduce experiments

**Fixed**: `src/preprocessing/windowing.py` + `augment_seizure_window`
- Added `random_seed` parameter (default: 42)
- Saves shift values to metadata
- Each augmented window has unique but deterministic seed
