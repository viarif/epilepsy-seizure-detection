# 方案1的批判性审查

## 🔍 发现的问题

### ❌ 问题1: 特征不平衡 - 小波特征严重不足
```
时域: 21个 (3通道 × 7)  
频域: 18个 (3通道 × 6)
小波: 6个  (1通道 × 6)  ← 为什么只用1个通道？
```

**问题**:
- 时域和频域都用了3个通道，但小波只用1个通道
- 逻辑不一致！如果3个通道都重要，为什么小波只提取1个？
- **论文1明确指出**: "Hjorth parameters and DWT coefficients were found to be the most informative features"

**应该**:
- **小波也应该从3个通道提取**: 3通道 × 6 = 18个小波特征
- 或者减少时域/频域通道数，保持一致性

---

### ❌ 问题2: 频段划分不符合论文
```
方案1的频段:
1. δ (4-8 Hz)
2. θ (8-13 Hz)
3. α (13-20 Hz)   ← 错误！
4. β (20-30 Hz)   ← 错误！
5. γ低 (30-50 Hz)
6. γ高 (50-100 Hz)
```

**问题**:
- **论文2明确使用**: 12个频段，每个4Hz宽 (0.25-48Hz)
- **论文3明确使用**: 8个频段 (δ: 4-8, θ: 8-13, β: 13-30, γ: 30-128)
- **标准EEG频段定义**:
  - α (alpha): **8-13 Hz**，不是13-20Hz！
  - β (beta): **13-30 Hz** ✓
  - 我们的α和θ重叠了！

**应该参考论文3的划分**:
```
1. δ (delta): 4-8 Hz
2. θ (theta): 8-13 Hz  
3. α (alpha): 这里有歧义，论文3把13-30Hz都归为β
4. β (beta): 13-30 Hz
5. γ1: 30-50 Hz
6. γ2-γ5: 50-128 Hz (分段)
```

---

### ❌ 问题3: 忽略了论文3的核心贡献 - 谱功率比

**论文3的关键发现**:
> "The key advantage of spectral power ratio features over the spectral power features is that certain ratio features are strong indicators"

**方案1只提取**:
- 绝对功率（6个频段 × 3通道 = 18个）
- **完全没有相对功率和功率比值！**

**论文3的特征结构**:
- 8个绝对功率
- 8个相对功率
- **28个功率比值** (C(8,2) = 28) ← 这是最有区分度的！

**应该**:
- 至少加入一些关键的功率比值，如 δ/α, θ/β 等
- 或者在特征选择阶段计算比值特征

---

### ❌ 问题4: 特征冗余 - Mean可能无用

**时域特征包含 Mean（均值）**:
```python
mean = np.mean(signal)  # 特征1
```

**问题**:
- EEG信号通常**已经去均值/高通滤波**
- 如果信号已预处理，mean ≈ 0，这个特征没有区分度
- 论文1没有明确列出"mean"作为关键特征

**应该**:
- 检查数据预处理流程，如果已去均值，移除这个特征
- 或者用更有意义的统计量，如**峰值因子**、**波形因子**

---

### ❌ 问题5: 小波分解层数不明确

```python
coeffs = pywt.wavedec(signal, 'db4', level=3)
```

**问题**:
- 分解到level 3，会产生：cA3, cD3, cD2, cD1 = 4个系数组
- 但描述说"6个小波特征"
- 数量对不上！

**应该明确**:
- 如果只取能量：4个特征
- 如果每层取2个统计量（均值+标准差）：4层 × 2 = 8个特征
- 或者取所有层的多个统计量

**论文1没有具体说明提取哪些统计量**，这是设计盲点

---

### ❌ 问题6: 总特征数与论文不一致

**方案1**: 45个特征 → Random Forest选12-16个

**论文对比**:
- **论文2**: 3通道 × 12频段 = 36特征 → 选8-10个
- **论文3**: 1-2通道 × 44特征 → 单特征或多特征
- **论文1**: 13个统计特征 + DWT → 没说具体数量

**问题**:
- 45个特征的规模在论文2和论文3之间
- 但**没有谱功率比特征**，这是论文3的核心
- 特征构成与任何一篇论文都不完全对应

---

### ⚠️ 问题7: 计算成本不均衡

```
时域特征: O(n)     - 每个窗口遍历一次
频域特征: O(n log n) - FFT
小波特征: O(n)     - 小波变换
```

**问题**:
- 只有1个通道做小波变换
- 但所有3个通道都做FFT
- 如果小波真的重要（论文1说它最informative），为什么计算投入最少？

**不一致的优先级**:
- 如果小波不重要 → 为什么要加？
- 如果小波重要 → 为什么只算1个通道？

---

### ❌ 问题8: 频率范围的选择没有依据

**方案1**: γ高 (50-100 Hz)

**论文说明**:
- **论文2**: 使用0.25-48Hz，去掉50Hz工频干扰
- **论文3**: 延伸到110-128Hz，但说明需要去除50Hz及谐波
- **实际EEG信号**: >50Hz通常是肌电噪声（EMG artifact）

**问题**:
- "γ高 (50-100 Hz)" 包含了50Hz工频干扰
- 论文明确说要排除50Hz附近 (47-53Hz, 97-103Hz)
- 这个频段可能带来更多噪声而非有用信息

**应该**:
- 要么不用>50Hz的频段
- 要么细分并排除工频干扰频段

---

## 🔧 改进建议

### 建议1: 修正频段划分（参考论文3）
```python
FREQUENCY_BANDS = {
    'delta': (4, 8),      # δ
    'theta': (8, 13),     # θ  
    'beta': (13, 30),     # β (合并α和β)
    'gamma1': (30, 50),   # 低γ
    'gamma2': (50, 70),   # 可选
    'gamma3': (70, 90),   # 可选
}
# 排除 47-53Hz (50Hz工频) 和 97-103Hz (100Hz谐波)
```

### 建议2: 统一通道策略
**选项A**: 所有特征都从3个通道提取
```
时域: 3通道 × 7 = 21
频域: 3通道 × 6 = 18
小波: 3通道 × 6 = 18
总计: 57个 → Random Forest选12-16
```

**选项B**: 减少冗余，只用2个通道
```
时域: 2通道 × 7 = 14
频域: 2通道 × 6 = 12
小波: 2通道 × 6 = 12
总计: 38个 → Random Forest选12-16
```

### 建议3: 加入谱功率比特征
```python
# 至少加入论文3提到的关键比值
key_ratios = [
    'delta/theta',
    'delta/alpha', 
    'theta/beta',
    'beta/gamma',
]
# 3通道 × 4比值 = 12个比值特征
```

### 建议4: 移除或替换Mean特征
- 如果信号已去均值 → 移除
- 否则 → 替换为**峰峰值 (peak-to-peak)** 或 **能量**

### 建议5: 明确小波特征提取
```python
# 明确提取方案
coeffs = pywt.wavedec(signal, 'db4', level=3)
# cA3, cD3, cD2, cD1

features_per_channel = [
    np.log(np.sum(coeffs[0]**2)),  # cA3 energy
    np.log(np.sum(coeffs[1]**2)),  # cD3 energy
    np.log(np.sum(coeffs[2]**2)),  # cD2 energy
    np.log(np.sum(coeffs[3]**2)),  # cD1 energy
]
# 每通道4个特征，不是6个
```

## 🎯 改进后的方案1+

### 特征构成
**时域** (21个): 3通道 × 7特征（移除Mean，加入峰峰值）
**频域** (18个): 3通道 × 6频段绝对功率
**频域比值** (12个): 3通道 × 4关键比值
**小波** (12个): 3通道 × 4小波能量

**总计**: 63个候选特征 → Random Forest选12-16

### 频段定义（参考论文3）
1. δ: 4-8 Hz
2. θ: 8-13 Hz
3. β: 13-30 Hz  
4. γ1: 30-50 Hz
5. γ2: 50-70 Hz (排除47-53Hz)
6. γ3: 70-90 Hz

### 关键比值（参考论文3发现）
1. δ/θ (DAR - Delta-Alpha Ratio的变体)
2. θ/β (Theta-Beta Ratio)
3. β/γ1 (Beta-Gamma Ratio)
4. (δ+θ)/(β+γ1) (Low/High frequency ratio)

## ✅ 总结

**方案1的主要问题**:
1. ❌ 小波只用1个通道，逻辑不一致
2. ❌ 频段划分错误（α频段定义错误）
3. ❌ 缺少谱功率比特征（论文3核心）
4. ❌ Mean特征可能无用
5. ❌ 小波特征数量不明确
6. ⚠️ 包含>50Hz频段但未处理工频干扰

**推荐改进**:
- 统一通道策略（3个通道都提取所有特征类型）
- 修正频段定义
- 加入关键谱功率比
- 明确小波特征提取方法
- 考虑去除或修正Mean特征
