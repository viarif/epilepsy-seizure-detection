# 当前标签策略详解

## 核心概念：两个阈值，两个用途

### 阈值1: 25% - 用于标签判定
**作用**: 决定一个窗口是否被标记为"癫痫"（label=1）

```python
overlap_ratio = 窗口内癫痫时长 / 窗口总时长

if overlap_ratio >= 0.25:  # 25%阈值
    label = 1  # 癫痫窗口
else:
    label = 0  # 正常窗口
```

### 阈值2: 50% - 用于增强选择
**作用**: 决定一个癫痫窗口是否可以被时间平移增强

```python
if label == 1 and overlap_ratio >= 0.50:  # 50%阈值
    # 这个窗口可以安全增强
    augmented_windows = time_shift(window, ±0.5s)
    # 增强后标签仍然准确
```

## 实际例子

假设癫痫发作时间: **2996秒 - 3036秒** (持续40秒)  
窗口大小: 4秒

### 例子1: 窗口 2994-2998秒
```
时间轴:  |----[2994___2996========2998]========3036----|
         正常   窗口开始  发作开始  窗口结束

overlap = 2998 - 2996 = 2秒
overlap_ratio = 2秒 / 4秒 = 50%

✓ overlap ≥ 25% → label = 1 (标记为癫痫)
✓ overlap ≥ 50% → 可以增强
```

**为什么可以增强？**
- 如果左移0.5秒: overlap ≈ 2.5秒 / 4秒 = 62.5% > 25% ✓
- 如果右移0.5秒: overlap ≈ 1.5秒 / 4秒 = 37.5% > 25% ✓
- 增强后标签仍然准确！

## 为什么需要两个阈值？

### 问题：如果用25%阈值增强会怎样？

**原始实现（有问题）**:
```python
for window, label in zip(windows, labels):
    if label == 1:  # 只要是癫痫窗口就增强
        augmented = time_shift(window, ±0.5s)
        y_list.extend([1, 1, 1])  # 假设增强后仍是癫痫
```

**问题案例**:
```
原始窗口 (overlap=26%):  |[癫痫]----------|  label=1 ✓

右移0.5秒后:             |--[癫痫]---------|  
                         overlap ≈ 13.75% < 25%
                         实际应该是 label=0
                         但我们标记为 label=1  ✗ 错误！
```

### 修复后的实现（当前版本）

```python
for window, label, overlap_ratio in zip(windows, labels, overlap_ratios):
    X_list.append(window)
    y_list.append(label)
    
    # 只增强overlap ≥ 50%的窗口
    if label == 1 and overlap_ratio >= 0.5:
        augmented = time_shift(window, ±0.5s)
        y_list.extend([1, 1])  # 增强后标签仍准确 ✓
```

**安全保证**:
```
overlap = 50% = 2秒

最坏情况（平移±0.5秒）:
- 剩余overlap = 2秒 - 0.5秒 = 1.5秒
- overlap_ratio = 1.5秒 / 4秒 = 37.5% > 25% ✓

结论: 只要原始overlap ≥ 50%，
      平移±0.5秒后仍能保证 ≥ 25%
      标签保持准确！
```

## 总结

**当前系统使用两个阈值**:

| 阈值 | 用途 | 值 |
|------|------|-----|
| **标签阈值** | 判断窗口是否为癫痫 | **25%** |
| **增强阈值** | 判断窗口是否可以增强 | **50%** |

**为什么这样设计？**

1. **25%标签阈值**: 提高敏感性，早期检测癫痫
   - 窗口只需1秒癫痫数据就标记为阳性
   - 符合临床早期预警需求

2. **50%增强阈值**: 保证增强后标签准确性
   - 防止时间平移导致标签错误
   - 确保数据质量

3. **实际效果**: 
   - chb01_03的21个癫痫窗口全部满足50%条件
   - 全部被增强，不平衡比改善到1:28.2
   - 没有任何标签污染问题

**这就是"原方案有风险，已修复"的含义！**
