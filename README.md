# 四通道跨患者癫痫窗口检测

当前采用原始轻量 CNN 方案，不做投票、平滑或患者适配。

最终评估改为 leave-one-subject-out：每折用 23 位患者训练，剩下 1 位患者测试。阈值固定为原验证集锁定的 logit `-2.632594108582`，不按测试患者重新选择。

项目用于科研实验，不是临床诊断或告警系统。

## 运行

推荐环境：

```powershell
$py = 'C:\Users\ASUS\anaconda3\envs\unet\python.exe'
```

复现 LOSO：

```powershell
& $py scripts/leave_one_patient_out.py `
  --device cuda:0 `
  --num-workers 4 `
  --resume
```

运行测试：

```powershell
& $py -m unittest discover -s tests -v
```

## 当前主要文件

| 文件 | 作用 |
|---|---|
| `scripts/leave_one_patient_out.py` | 当前最终 LOSO 评估入口。 |
| `scripts/train.py` | 原始 CNN 训练入口，保留用于生成基础 checkpoint 和阈值。 |
| `src/models/seizurenet_lite.py` | 2,991 参数轻量 CNN。 |
| `src/data/` | 窗口数据读取、层次阳性权重、hard-negative replay。 |
| `src/evaluation/metrics.py` | 窗口级、逐患者、发作级和误报指标。 |
| `results/final_report.md` | 当前 24 折 LOSO 总报告。 |
| `memory.md` | 详细方案、参数、历史实验和注意事项。 |
